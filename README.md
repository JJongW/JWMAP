# 오늘 오디가?

> 직접 가본 맛집만 수록하는 큐레이션 맛집 지도 + AI 추천 CLI

서울/수도권 맛집을 빠르게 결정하고 바로 출발할 수 있도록 돕는 서비스입니다.

---

## CLI 설치 및 사용법

터미널에서 바로 맛집/코스를 추천받을 수 있습니다.

### 설치

```bash
npm install -g odiga
```

### 사용법

```bash
# 맛집 추천
odiga "홍대 점심 뭐 먹지?"
odiga "강남 혼밥 맛집"
odiga "이태원 데이트 코스 짜줘"

# 통계 보기
odiga stats
```

### 추천 흐름

1. 검색어를 입력하면 AI가 맛집 또는 코스를 추천합니다
2. 목록에서 장소를 선택하면 상세 정보를 볼 수 있습니다
3. "다시 추천받기"를 선택하면 피드백을 남기고 새로운 추천을 받을 수 있습니다
4. 코스 모드에서는 선택한 코스를 저장할 수 있습니다

## 웹 서비스

### 주요 기능

- **검증된 장소만**: 운영자가 직접 방문하고 검증한 곳만 수록
- **자연어 검색**: "용산 혼밥 맛집" 같은 자연스러운 검색 지원
- **빠른 결정**: 한눈에 파악하는 카테고리와 필터
- **지도 앱 연동**: 탭 한 번으로 네이버/카카오 지도 앱 실행

### 검색 & 필터

- 지역별 필터링 (서울 19개 구역, 수도권)
- 카테고리별 필터링 (밥, 면, 카페, 술안주 등 10개 대분류)
- AI 자연어 검색 (Gemini 기반)

### 지도 연동

- 카카오 지도 통합 + 카테고리별 커스텀 마커
- 네이버/카카오 앱 딥링크
- 웹 지도 fallback
